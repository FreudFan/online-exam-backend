# online-exam-backend

#### 项目简介
online-exam-backend 是一个在线考试系统的后端模块。
基于 Jersey + SpringBoot 实现的的 restful 服务，主要包括用户管理、在线考试，自动批卷、成绩管理、错题管理、留言板、试卷管理、题库管理、试题科目维护等功能。
